fab-ops-yield-control-tower manufacturing ops control tower

Alarm triage, lot risk, release gates, and shift handoff for fab operations.

This public Pages site is a free summary surface for the repository. The full project combines a FastAPI backend with a browser control tower, but the core value is the workflow: grounded alarms, lot-at-risk prioritization, tool ownership, and signed next-shift handoff.

Operational story

  • Alarm queue linked to tool and lot context.
  • Recovery board with hold, watch, and release-ready lanes.
  • Tool ownership and escalation lane visibility.
  • Shift handoff plus signature proof for continuity.

Why this public version is static

  • The repo’s strongest signal is the reviewable workflow, not public backend hosting.
  • Static hosting keeps the portfolio accessible at zero cost.
  • The full backend remains available in-repo for local validation and future service deployment.
